How to Get Your Free API Key

API གསང་ལྡེ་ཇི་ལྟར་ལེན་དགོས།
What is an API Key?

An API key is like a password that allows our app to communicate with Google's AI. It's free for personal use and your key stays private in your browser.

Step 1: Go to Google AI Studio

Click the button below to open Google AI Studio in a new tab:

Open Google AI Studio

You'll need to sign in with your Google account (Gmail).

Step 2: Create a New API Key

Once you're in Google AI Studio, look for the "Create API Key" button.

Click this button:
+ Create API key

Step 3: Choose "Create API key in new project"

When asked about projects, you have two options:

Recommended

Create API key in new project

Choose this if you see "no projects available" or if this is your first time. Google will automatically create a new project for you.

Use existing project

Only choose this if you already have a Google Cloud project you want to use.

Don't have any projects? That's completely normal! Just select "Create API key in new project" and Google will set everything up for you automatically.

Step 4: Copy Your API Key

After Google creates your key, you'll see it displayed on screen. It will look something like this:

AIzaSyB-XXXXXXXXXXXXXXXXXXXXXXXXX Your key will be different

Click the copy button next to your key to copy it to your clipboard.

Keep your key private!

Don't share your API key publicly or commit it to GitHub. Treat it like a password.

Step 5: Paste in Sambhota Tutor

Go back to the AI Tutor page and paste your key in the input field:

Go to AI Tutor

Your key is stored only in your browser. We never see or store it on our servers.

Frequently Asked Questions

Is this really free?

Yes! Google offers a free tier for the Gemini API that's more than enough for personal learning use. You won't be charged unless you exceed very high usage limits (which is unlikely for individual learners).

Why do I need my own API key?

Each person having their own key ensures the service remains free and sustainable. If we used a shared key, it would quickly hit usage limits and cost money to maintain.

Is my data safe?

Your API key stays in your browser's local storage. Your questions go directly to Google's servers, not ours. We don't collect or see your conversations.

What if I see "no projects available"?

That's fine! Just click "Create API key in new project" and Google will automatically create a project for you. You don't need to set up anything on Google Cloud Platform beforehand.

Can I use this key for other projects?

Yes! Once you have a Gemini API key, you can use it for any project that supports the Gemini API.

Still having trouble?
