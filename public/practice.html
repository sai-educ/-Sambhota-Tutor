ཀ ka - Consonant 1 of 30
On path - Good!

How to Practice

  1. Select a character from the left panel
  2. Follow the numbered points and arrows
  3. Draw each stroke in order (1, 2, 3...)
  4. Green = correct path, Red = off path
  5. Tap "Clear" to try again
